原子核结构

原子核位于原子中心，由质子和中子组成。质子带正电，中子不带电。

原子序数 Z = 质子数
质量数 A = 质子数 + 中子数
例: ¹²C 表示碳-12，Z=6，A=12

核力与核能

  • 强核力：将质子和中子紧密束缚在原子核内
  • 短程性：仅在约10⁻¹⁵米的距离内起作用
  • 结合能：将核子结合成原子核时释放的能量

同位素

同位素是质子数相同、中子数不同的原子。例如氢的三种同位素：

氕 ¹H
0个中子
氘 ²H
1个中子
氚 ³H
2个中子

电子排布规则

能层与2n²法则

K层 (n=1)
最多2电子
L层 (n=2)
最多8电子
M层 (n=3)
最多18电子
N层 (n=4)
最多32电子

排布原理

  • 能量最低原理：电子先填能量低的轨道
  • 泡利不相容原理：每个轨道最多2电子，自旋相反
  • 洪特规则：简并轨道电子先单独占位，自旋平行
碳 C: 1s² 2s² 2p²

价电子

最外层电子决定元素的化学性质。例：钠(Na)的3s¹电子易失去，氟(F)的2p⁵电子易获得。

离子键形成过程

离子键是金属原子失去电子、非金属原子获得电子，通过静电作用结合而成的化学键。

Na + Cl → NaCl

钠失1电子，氯得1电子

Na + F → NaF

氟是电负性最强的元素

K + Cl → KCl

钾比钠更易失去电子

Mg + Cl₂ → MgCl₂

镁失2电子，需2个氯原子

1

接近

钠原子和氯原子相互靠近，外层电子开始相互作用。

2

电子转移

钠原子最外层的1个3s¹电子转移到氯原子外层，钠变为Na⁺，氯变为Cl⁻。

3

离子键形成

正负离子之间的静电吸引力（库仑力）将它们结合成离子化合物。

Na → Na⁺ + e⁻ (电离)
Cl + e⁻ → Cl⁻ (亲和)
Na⁺ + Cl⁻ → NaCl ↓ (离子键)

离子键特点

  • 形成离子化合物（如食盐NaCl）
  • 固态为晶体结构，熔沸点较高
  • 可溶于水，水溶液导电
  • 方向性、饱和性较弱

共价键形成过程

共价键是两个原子各提供电子，形成共用电子对，将原子结合在一起的化学键。

H + H → H₂

各贡献1电子，形成1对共用电子

O + O → O₂

形成1个σ键 + 1个π键（双键）

N + N → N₂

形成1个σ键 + 2个π键（三键）

O + 2H → H₂O

氧与两个氢分别形成共价键

1

轨道接近

两个原子的价电子轨道相互接近。

2

轨道重叠

轨道发生头碰头(σ键)或肩并肩(π键)的重叠，电子在两核间运动。

3

共用电子对

两个原子各提供1个电子形成共用电子对，同时被两个原子核吸引。

H· + ·H → H:H (或 H–H)
共用电子对: 1对 = 单键
共用电子对: 2对 = 双键
共用电子对: 3对 = 三键

共价键类型

  • σ键(σ)：沿键轴方向头碰头重叠，稳定
  • π键(π)：肩并肩重叠，较不稳定
  • 单键：1个σ键，如C–C
  • 双键：1个σ键+1个π键，如C=C
  • 三键：1个σ键+2个π键，如C≡C
  • 极性键：电子对偏向电负性大的原子
  • 非极性键：电子对不偏移

元素周期表

元素周期表按原子序数递增排列，化学性质相似的元素在同一族。电子层结构决定了元素在周期表中的位置。

周期与能层

  • 第1周期：2种元素 (H, He) - 只有K层
  • 第2周期：8种元素 (Li ~ Ne) - L层填充
  • 第3周期：8种元素 (Na ~ Ar) - M层填充
  • 第4周期：18种元素 (K ~ Kr) - N层开始填充
  • 第5周期：18种元素 (Rb ~ Xe)
  • 第6周期：32种元素 (Cs ~ Rn)
  • 第7周期：不完全周期

族与价电子

  • 第1族：1个价电子 - 碱金属（氢除外）
  • 第2族：2个价电子 - 碱土金属
  • 第13-17族：3-7个价电子
  • 第18族：8个价电子（He为2个）- 稀有气体

分区

  • s区：第1-2族，ns¹-²
  • p区：第13-18族，ns²np¹-⁶
  • d区：第3-12族，(n-1)d¹-¹⁰ns²
  • f区：镧系和锕系
